# Akwaaba AI — Marketing Website

AI-powered business communication platform for Ghana. Built with Next.js 14+, TypeScript, and Tailwind CSS.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Stack

- **Framework**: Next.js 14+ (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Deploy**: Vercel

## Pages

- `/` — Homepage
- `/features` — Feature breakdown
- `/pricing` — 5 pricing tiers
- `/use-cases` — Industry solutions
- `/about` — Company story
- `/contact` — Contact form
- `/api-docs` — API documentation landing page
- `/blog` — Blog (coming soon)
- `/privacy` — Privacy Policy
- `/terms` — Terms of Service
- `/cookie-policy` — Cookie Policy
- `/login` — Redirects to app
- `/signup` — Redirects to app
